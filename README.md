## 규칙

  - `master branch` 를 직접 수정하는 것이 아니라 자신의 컴퓨터에 레포지토리를 `clone` 한 후 개발을 진행한 후 자신의 `branch`에 `push` 해야합니다.  

  - `master branch` 로 `merge` 하는 것은 매일 저녁에 다른 팀원들의 확인 후에 진행하는 것으로 하겠습니다😁😁.
